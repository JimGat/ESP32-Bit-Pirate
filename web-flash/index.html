Requirements

Browser: Chrome, Edge, or Brave on desktop (WebUSB is required).
Cable: Use a data-capable USB-C cable, not a charge-only one.
Boot mode: Some ESP32-S3 boards need a quick BOOT+RST tap to enter the USB bootloader if the device isn't detected.

About this flasher

This is the firmware flasher for the JimGat/ESP32-Bit-Pirate AI-ready fork — built around the upstream ESP32-Bit-Pirate multi-tool firmware with an added direct HTTP/JSON automation API designed for AI agents (JARVIS, Claude Code, etc.).

The fork remains platformio + Arduino IDE compatible with upstream, so contributions flow back cleanly. The web flasher ships the built firmware for the selected board so you can go from unboxing to hacking in under a minute.
