Web AVR Programmer help

    

      
What it does

      
Runs AVRDUDE in the browser to detect AVR chips, read Flash, write firmware, verify images and perform chip erase operations.

    

    

      
Adapters

      
    
        
  • Bus Pirate Raw SPI/ISP is the main profile for in-system programming.
    • 
        
  • AVR109 and other serial bootloader profiles work when the target already has a compatible bootloader.
    • 
        
  • For ISP, connect MOSI, MISO, SCK, RESET, GND and a voltage reference that matches the target.
    •
