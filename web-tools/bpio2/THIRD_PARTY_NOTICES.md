## BPIO2 FlatBuffers interface

This web tool implements the BPIO2 v2 wire protocol using the public schema and documentation from:

- https://github.com/DangerousPrototypes/BusPirate-BPIO2-flatbuffer-interface
- https://docs.buspirate.com/docs/binmode-reference/protocol-bpio2/

The reference BPIO2 tooling and schema are distributed under the Apache License 2.0.
This web implementation is a small browser-oriented codec and does not bundle the generated TypeScript tooling.
