Web BPIO2 Bit Bang I/O Controller help

    

      
What is it for?

      
Control and monitor GPIOs, scan and communicate with I2C devices, exchange SPI data and build simple hardware sequences from the browser through a BPIO2-compatible adapter.

    

    

      
Adapter

      
    
        
  • Use a BPIO2-compatible device exposed as a Web Serial port.
    • 
        
  • With ESP32 Bit Pirate, use mode USB → adapter → BPIO2 GPIO/SPI/I2C, assign the eight GPIOs and let the device reboot.
    • 
        
  • SPI uses IO0=CS, IO1=SCK, IO2=MOSI and IO3=MISO, with preset or custom clock speeds up to 40 MHz. I2C uses IO1=SCL and IO2=SDA.
    • 
      

    

    

      
Connection

      
    
        
  1. Connect target ground to adapter ground and keep signals within the supported voltage limits.
    2. 
        
  2. Click Connect and select the BPIO2 serial interface.
    3. 
        
  3. Use GPIO for live monitoring and direct control, I2C or SPI for bus transactions, and Sequence for simple non-real-time automation.
