Web ESP Tool help

    

      
What it does

      
Flash, read, erase and merge firmware images for ESP8266, ESP32 and newer ESP variants through the ROM serial bootloader.

    

    

      
Adapter

      
    
        
  • Use a USB serial adapter or a development board with built-in USB-UART.
    • 
        
  • Connect TX to RX, RX to TX and share GND. Keep 3.3 V logic levels.
    • 
        
  • For bare modules, wire EN/RESET and BOOT/GPIO0 so the chip can enter download mode.
    • 
      

    

    

      
Connection

      
    
        
  1. Hold BOOT/GPIO0 low, reset or power-cycle the ESP, then release reset.
    2. 
        
  2. Click Connect and select the serial port.
    3. 
        
  3. Load firmware files, choose the matching layout and run the operation.
    4.
