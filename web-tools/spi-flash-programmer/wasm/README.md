# rflasher WASM integration

This directory is reserved for the rflasher WebAssembly engine or loader artifacts used by the ESP32 Bit Pirate Web SPI Flash Programmer.

`ArthurHeymans/rflasher` is licensed GPL-2.0-or-later. When the WASM engine is added here, include the matching generated files, upstream notices, license text, and corresponding modified Rust source or a clear source offer/link for the exact revision used.

The current prototype does not vendor rflasher source or compiled WASM yet. It validates the ESP32 Bit Pirate serprog adapter through a small Web Serial diagnostic layer first.
