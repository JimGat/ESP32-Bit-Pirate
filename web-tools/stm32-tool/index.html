STM32 Web Tool help

    

      
What it does

      
Identify, back up, erase and flash STM32 devices either through the factory UART bootloader or through an ST-Link probe over SWD.

    

    

      
Adapters

      
    
        
  • UART mode: use a 3.3 V USB serial adapter connected to the STM32 bootloader UART pins.
    • 
        
  • ST-Link mode: use ST-LINK/V2 or ST-LINK/V2-1 with SWDIO, SWCLK, GND and target voltage sense connected.
    • 
        
  • ST-Link uses WebUSB, so use Chrome, Edge or another Chromium browser. Firefox is not supported for this mode.
    • 
      

    

    

      
Connection

      
    
        
  1. For UART, set BOOT0 high, reset the target, then click Connect.
    2. 
        
  2. For ST-Link, power the target, connect SWD, then click Connect and select the probe.
    3. 
        
  3. Read a backup before writing when working on hardware you cannot easily recover.
